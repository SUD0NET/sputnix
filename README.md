# sputnix
"Soviet" Operating System 

![sputnix](./SPUTNIX.png)

To setup the build tools, use:
```
chmod +x setup.sh
./setup.sh
```

To build the os, use:
```
chmod +x build.sh
./build.sh
```

To run using qemu, use:
```
chmod +x run.sh
./run.sh
```
